## Key Takeaways

1. **Image Classification**: Core CV task - assign labels to images
2. **Semantic Gap**: Humans see objects, computers see numbers
3. **K-NN**: Simple, no learning, but slow at test time
4. **Linear Classifier**: f(x,W) = Wx + b, learns templates
5. **Limitations**: Single template per class is too restrictive
